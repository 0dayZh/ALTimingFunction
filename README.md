# ALTimingFunction

Smooth animation timing function for iOS && OS X in Objective-C.

## Install

```
pod 'ALTimingFunction'
```

## Supported Timing Function

- [x] Circ Easing
- [x] Cubic Easing
- [x] Expo Easing
- [x] Quad Easing
- [x] Quart Easing
- [x] Quint Easing
- [x] Sine Easing
- [x] Back Easing
